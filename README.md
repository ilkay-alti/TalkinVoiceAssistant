# Python Talking Voice Assistant

A silly talking assistant app (upgradeable)

## Installation


```bash
git clone https://github.com/ilkay-alti/TalkinVoiceAssistant.git
pip3 install VoiceAsistant-ilkayus
```

## Start Assistant

```bash
python ana.py
```

## Voice Command
